## Wk3 Smart Chair

### ESP8266, RGB LED Module, Touch sensor, Photoregistor sensor 
**RGB LED Module** connects to: R -> D2, G -> D3, B -> D4 , GND -> - rail \
**Touch sensor** SIG -> D5 \
**Photoregistor** SIG -> A0
